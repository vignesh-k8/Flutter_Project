# To-Do UI Flutter
A simple task manager UI built in DartPad using Flutter.  
This project shows how to add and remove tasks dynamically using a TextField and buttons.

## Features
- Add tasks manually via TextField
- Delete tasks with a trash icon
- Clean layout using Material 3
- UI-only demo (no backend)

## How to Run
1. Go to https://dartpad.dev
2. Switch to Flutter mode (top-left dropdown)
3. Paste the code from To-Do_List.dart
4. Click Run to see the app
